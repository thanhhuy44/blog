### Installation

1. Clone the repo

```sh
git clone https://github.com/thanhhuy44/blog.git
```

2. Start Back-end server

```sh
cd back-end
yarn && yarn start
```

3. Start Front-end

```sh
cd front-end
yarn && yarn dev
```

```.env
DB_URL='your mongoDB url'
PRIVATE_KEY='your private key'
```
